Description

What is this?

The Rick and Morty API is a REST(ish) and GraphQL API based on the television show. You will have access to about hundreds of characters, images, locations and episodes. The Rick and Morty API is filled with canonical information as seen on the TV show.

Who are we?

We are Axel Fuhrmann, a guy who likes to develop things and Talita, the "Rick and Morty data scientist" and hardcore fan.

Why did we build this?

Because we were really interested in the idea of writing an open source project and also because Rick and Morty is our favorite show at that moment, so why not?
